Setting up your Northwestern website

This page will walk you through the basics of getting your math.northwestern.edu website up and running.
This is an extremely barebones guide that I wrote in a hurry so if you have questions or need help debugging feel free to email me or drop by my office.

My website, for instance, is here.
If your math username were nilay this guide would set your webpage up at http://www.math.northwestern.edu/~nilay.
Note: your math username should be the username in your northwestern math email address (something like ...@math.northwestern.edu).

The website is written in HTML, which you can write in any plaintext editor of your choice. I'll briefly explain how to link to pdfs and add images.
(There are a lot of resources on HTML and related web dev stuff, so just google around if you want to learn more.)
Then the HTML is uploaded to your math account (on a linux machine somewhere) via ftp to the appropriate folder, after which you should be able to view it in your browser.

Prerequisites

Here's what you need to get started:
  • an HTML file containing the contents of your webpage. Here is a sample to get started.
  • a text editor to edit the HTML file. Notepad and TextEdit will do fine.
  • a tool to transfer your HTML file to the northwestern math server.
    There are many programs you can use, but one such is Filezilla. Download and install the client appropriate for your operating system.
  • important: make sure to follow this guide on a computer connected to a northwestern network! Alternatively you can VPN into the northwestern network (instructions for this are here).
    Otherwise, due to recent security changes, the northwestern servers will not accept your connection.

Guided setup

First we'll download the sample HTML file provided in the prerequisites section above.
To do this click on that link and then right click anywhere in your browser and hit view page source (the exact wording may differ based on your browser).
Now copy all the code into your text editor. Save the file in your working directory as index.html.
You could instead just download the source, but I find that the formatting gets messed up sometimes.
Now open the HTML file in a browser (probably just by double clicking the file). You should see the sample webpage.

Since you're probably not Alexander Grothendieck, let's change the page heading
In your text editor change the text within the h1 tags in the code from Alexander Grothendieck to your name and save the file. Keep the text editor open for now.
Hit refresh in your browser. The webpage should now show your name as the header.
Notice that the text in your browser's tab is still wrong! This can be fixed by editing the title tag in the code similar to before.



There are two things left to fix. The first is that the example link to the pdf won't work. That's because the pdf file is missing.
Place any pdf file in the same directory as the HTML file. I'll put in paper.pdf. The code needs to be changed to point to the correct pdf.
Change the text within the quotes after the a href= to paper.pdf, save, refresh, and the link should now work!



I'll leave you to figure out how the external website link works. Next we need to fix the broken image.
Again place any image file in the same directory as the HTML file. I'll put in portrait.jpg. Change the code to point to the correct image as follows.
Change the text within the quotes after the img src= to portrait.jpg. As usual, save and refresh, and your image should now appear.



That's all say about building your webpage. Currently the webpage is stored and viewed locally on your computer.
The next step is to upload your files to the math server. The server will then serve those files to anyone who visits the appropriate web address.
To do this open up Filezilla. There will be two panes. The left represents files on your computer. On the left, navigate to the directory with your website's files.
Now we need to connect to the math server. Again make sure you're connected to the Northwestern network directly or through VPN.
Near the top of Filezilla, in the Host field enter lunt.math.northwestern.edu.
For username and password enter your math username and password. Finally, for port, enter 22.
Hit Quickconnect and you should see something like this:

The right hand pane is displaying the files on the math server. Yours may look different than mine.
In the right hand pane , if it doesn't exist already, create a directory called public_html by right clicking the folder labelled by your username and choosing create directory.



Now select your files in the left hand pane, right click them, and hit upload. You should see something like:

although I uploaded into a folder called temp instead of public_html, since I didn't want to replace the files I already had.
Now, in your browser, head to www.math.northwestern.edu/~USERNAME, replacing USERNAME with your math username. The website should appear!
